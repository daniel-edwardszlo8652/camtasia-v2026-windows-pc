## Overview

Camtasia is a Windows PC application centered on screen recording and video editing. It keeps recording and post-production in the same desktop tool, so you can go from captured footage to a finished export without jumping between separate apps.

This repository presents a desktop-first release that includes LUT handling and export presets. Together, these options give you more control over visual output and help tailor rendered videos for different delivery needs.
